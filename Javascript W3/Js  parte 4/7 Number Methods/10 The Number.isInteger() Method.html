JavaScript Numbers

The isInteger() Method

The isInteger() method returns true if the argument is an integer.

Otherwise it returns false.

Safe integers are all integers from -(253 - 1) to +(253 - 1). This is safe: 9007199254740991. This is not safe: 9007199254740992.

Note The Number methods Number.parseInt() and Number.parseFloat() are the same as the Global methods parseInt() and parseFloat(). The purpose is modularization of globals (to make it easier to use the same JavaScript code outside the browser).
